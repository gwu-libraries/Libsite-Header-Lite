Example Page

This is just an example of how the header might look in a simple page. The code for this page (page.html) has comments and examples of how to call the css files or embed the css in the page.

The header is fluid/responsive (note: gw logo drops out under 720px)

Files for the simple Libsite7 Lite Header:

  • libheader7_lite.html
  • libfooter7_lite.html
  • libheader7_lite.css (includes the main Libraries site styling for content)
  • libheader7_lite_ie.css (ie fix for responsive: min-width 720px)
  • (optional) libheader7_lite_print.css (print-friendly css for media='print')
  • (optional) the favicon.ico file is included
  • (variant) libheader7_lite_dropin.css (for paste-in code in vendor tools like Libguides)

If you have to include the css in the <head> or inline as 'style=' give #libheader and #libfooter a min-width of 720px.

Files for the Bootstrap Libsite7 Lite Header (example page with more instructions)

  • bootstrap.html (html scaffolding with comments)
  • libheader7_lite.css
  • libheader7_lite_bootswatch.css
  • libheader7_lite_ie.css (fix for older versions of IE)
  • javascript files in the JS directory
  • images in the IMG directory
  • bootstrap css files (bootsrap.css and bootsrap-responsive.css, you can use the ones here in the repo or better yet, get the latest from http://twitter.github.com/bootstrap/index.html)
