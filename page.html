Example Page

This is just an example of how the header might look in a simple page. The code for this page (page.html) has comments and examples of how to call the css files or embed the css in the page.

The header is fluid/responsive (note: gw logo drops out under 720px)

Files you will need:

  • libheader7_lite.html
  • libfooter7_lite.html
  • libheader7_lite.css (includes the main Libraries site styling for content)
  • libheader7_lite_ie.css (ie fix for responsive: min-width 720px)
  • (optional) the favicon.ico file is included

If you have to include the css in the <head> or inline as 'style=' give #libheader and #libfooter a min-width of 720px.
