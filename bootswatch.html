Bootstrap Libsite7 Lite Swatch

What is this? This is a modified "swatch" for Twitter Bootstrap http://twitter.github.com/bootstrap/index.html. You can use the basic html structure of this page and the Bootstrap files to deploy responsive, branded pages/sites for GW Libraries

Implementation

The main file you will need is the HTML scaffold file ('bootswatch.html'). All of the necessary code is explained there.

  • CSS: call libsite7_lite.css and libheader7_lite_bootswatch.css. Include both files (in order) in the <head> after the bootstrap css link. Why 2 files? Because the first one is also used for the "simple" header
  • HTML Scaffold: bootswatch.html (this page). This is the html template with lots of comments
  • Internet Explorer Fixes: include the libheader7_lite_ie.css file after the rest of the css. Also add the html5shiv.js file to your JS directory (this file, borrowed from the Bootstrap starter template, helps retro fix older versions of IE)

Layout options for Twitter Bootstrap (scaffolding) are outlined here: http://twitter.github.com/bootstrap/scaffolding.html
